# web by vaazee

A collection of static HTML pages, hosted on GitHub Pages.

## Structure

Each category has its own folder. Drop an HTML file into a folder and it's live at
`https://vaazee.github.io/web-by-vaazee/<category>/<file>.html`.

- `random/` 🎲
- `music/` 🎵
- `movies/` 🎬
- `tv/` 📺
- `piano/` 🎹
- `tech/` ⚙️
- `coding/` 💻
- `books/` 📚
- `chess/` ♟️

The root `index.html` links to every category. Each category's `index.html` is a landing page.

## Hosting

Served from the `main` branch root via GitHub Pages. `.nojekyll` disables Jekyll so files
are served exactly as written.
